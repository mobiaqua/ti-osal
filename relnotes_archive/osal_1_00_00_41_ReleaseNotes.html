Version Information

This product's version follows a version format, M.mm.pp.bb, where M is a single digit Major number, mm is 2 digit minor number, pp is a 2 digit patch number, and b is an unrestricted set of digits used as an incrementing build counter.

To support multiple side-by-side installations of the product, the product version is encoded in the top level directory, ex. osal_1_00_00_41.

Subsequent releases of patch upgrades will be identified by the patch number, ex. OSAL 1.00.01.XX with directory osal_1_00_01_XX. Typically, these patches only include critical bug fixes.
